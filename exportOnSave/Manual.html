Export on save

A krita plugin to help export from .kra to .png automatically on save This will help game dev folks where you can save in Krita, then have your asset ready immediately on your favorite game engine as .png format

Usage

Just import and enable this plugin to use. Note that the export settings are as follow
  • alpha=True
  • compression=3
  • indexed=False
This plugin is only tested on Krita(x64) Windows version 5.2.0
